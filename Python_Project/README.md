# 🍱 Swiggy Sales Analysis

Exploratory data analysis of food delivery orders across Indian cities.  
Dataset: ~197,000 records · 10 columns.

## Tools
Python · Pandas · Matplotlib · Seaborn · Plotly

## Files
- `SWIGGY_SALES_ANALYSIS.ipynb` — full analysis notebook
- `swiggy_data.xlsx` — source dataset

## What's Inside
KPIs: Total Sales, Total Orders, Average Order Value, Average Rating, Rating Count

Charts: Monthly and daily sales trends, quarterly performance, top 5 cities by sales, sales split by food type (Veg vs Non-Veg)

# 🍱 Аналіз продажів Swiggy

Розвідковий аналіз замовлень сервісу доставки їжі по містах Індії.
